# Herpath - Opportunity Discovery Platform

A modern SaaS platform for professional opportunity matching and application preparation.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```

## Tech Stack
- **Framework**: React 19 + TypeScript
- **Bundler**: Vite
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Routing**: React Router DOM

## Features
- **Landing Page**: Premium hero section and features overview.
- **Dashboard**: Stats and top matches.
- **Opportunity List**: Filterable job/opportunity feed.
- **Opportunity Detail**: AI-powered match score breakdown.
- **Document Builders**: Real-time editors for CVs, Cover Letters, and Proposals with live A4 preview.
- **Onboarding**: Multi-step profile setup.
